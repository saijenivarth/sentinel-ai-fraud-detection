# # # Sentinel AI - Real-Time Fraud Detection System
A system that:  detects suspicious transactions blocks high-risk payments gives fraud probability score 
## Tech Stack:
Frontend: React + Tailwind
Backend: FastAPI
ML: Scikit-learn
Database: SQLite/PostgreSQL## Features: 
- Real-time transaction monitoring
- AI fraud prediction
- Risk scoring
- Analytics dashboard
- JWT authentication
- Live transaction simulation<img width="1085" height="786" alt="image" src="https://github.com/user-attachments/assets/4206b6dd-2663-44f8-9e84-62bf33110903" />
## Installation
-cd frontend
-npm install
-npm run dev
## Backend:
-pip install -r requirements.txt
